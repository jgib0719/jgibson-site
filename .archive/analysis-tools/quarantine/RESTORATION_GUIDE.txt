HOW TO RESTORE DATA
-------------------

Option 1: Restore Individual Section
Run the appropriate restoration script:

    python3 reinsert_section1_modal_data.py    # Restore Network Fundamentals
    python3 reinsert_section2_modal_data.py    # Restore Network Access
    python3 reinsert_section3_modal_data.py    # Restore IP Connectivity
    python3 reinsert_section4_modal_data.py    # Restore IP Services
    python3 reinsert_section5_modal_data.py    # Restore Security Fundamentals
    python3 reinsert_section6_modal_data.py    # Restore Automation

Option 2: Restore All Sections (Bulk)
Run all restoration scripts in sequence:

    for i in {1..6}; do python3 reinsert_section${i}_modal_data.py; done

Option 3: Manual Restoration from Backups
Copy backup files to restore original state:

    cp section1_backup.html ../section1.html
    cp section2_backup.html ../section2.html
    cp section3_backup.html ../section3.html
    cp section4_backup.html ../section4.html
    cp section5_backup.html ../section5.html
    cp section6_backup.html ../section6.html

RESTORATION PROCESS DETAILS
----------------------------

Each restoration script:
1. Creates a new backup of the current (modified) file
2. Reads the extracted modal data from the JSON file
3. Reconstructs the original topicDetails JavaScript object
4. Replaces the placeholder comment with the full data
5. Writes the restored content back to the original file
6. Confirms successful restoration

Safety Features:
- Automatic backup creation before any modification
- Validation of data integrity during restoration
- Error handling for missing files or corrupted data
- Confirmation messages for successful operations

TROUBLESHOOTING
---------------

If restoration fails:
1. Check that you're in the quarantine folder: /var/www/jgibson.site/public_html/quarantine
2. Verify the JSON data files exist and are readable
3. Ensure the target HTML files are writable
4. Use the backup files as a fallback: cp section*_backup.html ../

For complete system reset:
1. Copy all backup files to parent directory
2. Restart from original state
3. Re-run extraction if needed
